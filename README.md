### **Vanadium**

Простой архиватор

**Версия приложения: 1.0**

## 🛠️ Технологии и зависимости

Проект разработан с использованием следующих технологий:

-   **[Python 3.14](https://www.python.org/)** — основной язык программирования.  
    Распространяется по лицензии [PSF License v2](https://docs.python.org/3/license.html).
-   **[PyQt6](https://www.riverbankcomputing.com/software/pyqt/)** — фреймворк для создания графического интерфейса.  
    Распространяется под лицензией [GNU GPL v3](https://www.gnu.org/licenses/gpl-3.0.html) (или коммерческой лицензией Riverbank Computing).
-   **Стандартные библиотеки Python** (`zipfile`, `tarfile`, `ctypes` и др.) — входят в состав дистрибутива Python и распространяются по лицензии PSF.

> ⚠️ **Важно:** Поскольку PyQt6 распространяется под GNU GPL v3, при распространении Vanadium Archiver необходимо соблюдать условия этой лицензии (например, предоставлять исходный код производных работ), если вы не приобрели коммерческую лицензию у Riverbank Computing. Сам Vanadium Archiver остается под лицензией MIT, но его использование вместе с PyQt6 накладывает дополнительные ограничения на дистрибуцию бинарных файлов.
